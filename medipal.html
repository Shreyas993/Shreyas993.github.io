Project Overview

MEDIPAL is a healthcare-focused web platform designed to improve patient-doctor interaction through a structured and user-friendly frontend experience.

The platform focuses on usability, appointment workflows, interface accessibility, and organised healthcare navigation while maintaining a clean and modern UI structure.

Core Features

Patient Interface

User-friendly patient dashboard with accessible navigation and structured workflows.

Doctor Interaction

Designed workflows enabling efficient communication between patients and healthcare providers.

Appointment Flow

Structured scheduling interface for streamlined healthcare appointment management.

Responsive UI Design

Modern frontend layout focused on usability, accessibility, and clean visual hierarchy.
