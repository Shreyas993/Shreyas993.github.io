Project Overview

This project captures and analyses live network traffic using packet inspection techniques. The platform identifies traffic patterns, classifies protocols, and visualises network behaviour in real-time.

The system integrates both CLI and GUI dashboards to provide a professional cybersecurity monitoring experience.

Core Features

Live Packet Capture

Captures and analyses live packets with configurable filters.

Protocol Classification

Detects TCP, UDP, ICMP, and ARP traffic automatically.

Real-Time Dashboard

Displays packet statistics, traffic distribution, and runtime analytics visually.

Anomaly Detection

Detects abnormal or oversized packets during monitoring.
