Project Overview

This project was developed to monitor and analyse live network traffic in real-time using packet inspection techniques. The system captures, classifies, and visualises packets while identifying abnormal traffic behaviour and generating useful analytical insights.

The tool integrates both CLI and GUI dashboards to provide a professional monitoring experience for network analysis and cybersecurity-focused investigations.

Core Features

Live Packet Capture

Captures live network packets with configurable protocol filters.

Protocol Classification

Analyses and classifies TCP, UDP, ICMP, and ARP traffic.

Real-Time Dashboards

Displays packet statistics, alerts, and traffic distribution visually.

Anomaly Detection

Detects unusually large packets and suspicious traffic behaviour.
