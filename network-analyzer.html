Real-Time Network Traffic Analyzer

Detailed Project Overview

Overview

This project is a real-time network monitoring tool designed to capture, analyse, and visualise live packet-level traffic. It focuses on identifying abnormal patterns and providing insights into network behaviour.

Key Features

  • Live packet capture with protocol filtering
  • Protocol classification (TCP, UDP, ICMP, ARP)
  • Real-time CLI dashboard with alerts
  • GUI dashboard with visual charts
  • CSV logging and traffic analysis

Tools & Technologies

Python, Scapy, Matplotlib, Tkinter, Colorama
