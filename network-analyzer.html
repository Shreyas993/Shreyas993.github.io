Project Overview

This project was developed to monitor and analyse live network traffic in real-time using packet inspection techniques. The system captures, classifies, and visualises packets while identifying abnormal traffic behaviour and generating useful analytical insights.

The platform integrates both CLI and GUI dashboards to provide a professional monitoring experience for cybersecurity analysis and network traffic investigations.

Core Features

Live Packet Capture

Captures and analyses live network packets with configurable protocol filters and traffic monitoring support.

Protocol Classification

Detects and classifies TCP, UDP, ICMP, and ARP traffic for structured network analysis.

Real-Time Dashboards

Displays packet statistics, traffic distribution, runtime metrics, and monitoring insights visually.

Anomaly Detection

Detects unusually large packets and suspicious traffic behaviour during monitoring sessions.
