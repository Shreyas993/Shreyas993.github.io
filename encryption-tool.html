Project Overview

This project was developed to securely encrypt and decrypt files while maintaining file integrity and audit logging. The system implements symmetric encryption using Fernet and generates SHA-256 hashes for integrity verification.

The tool also maintains activity logs and hash records, supporting secure workflows and forensic traceability during file operations.

Core Features

File Encryption

Encrypts files securely using AES-128 symmetric encryption.

Integrity Verification

Generates SHA-256 hashes to verify file integrity and detect tampering.

Forensic Logging

Tracks encryption and decryption activities through structured logs.

Error Handling

Handles invalid keys, corrupted files, and missing file scenarios safely.
