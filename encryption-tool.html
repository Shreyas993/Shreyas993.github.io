Secure File Encryption Tool

Detailed Project Overview

Overview

Developed a secure file encryption system implementing AES-128 encryption using Fernet. Integrated SHA-256 hashing to ensure file integrity and added logging mechanisms for auditing and forensic tracking of file operations.

Key Features

  • Encryption and decryption using symmetric key cryptography
  • SHA-256 hashing for integrity verification
  • Activity logging for auditing and traceability
  • Error handling for invalid keys and corrupted files
  • User-friendly CLI with structured interface

Tools & Technologies

Python, Fernet (AES-128), SHA-256, Colorama
